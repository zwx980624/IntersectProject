## 二、解题思路描述

### 题目需求简述

- 题目需求为，给定若干直线，求其交点个数
- 直线条数`1000 <= N <= 500000`
- 交点个数`0 <= h <= 5000000`
- 运行时长60s

## 三、使用方法

- 进入bin/

- 运行`intersect.exe -i input.txt -o output.txt`

示例：

```bash
git clone https://github.com/zwx980624/IntersectProject.git
cd bin
intersect.exe -i ..\test\test1.txt -o test1.out
```
